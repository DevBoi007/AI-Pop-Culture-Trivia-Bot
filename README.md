# 🌍 AI Culture Trivia Chatbot

An intelligent, AI-powered trivia chatbot that tests your knowledge of world cultures — history, festivals, food, geography, and languages. Built with Python, Streamlit, Google Gemini AI, and SQLite.

---

## ✨ Features

| Feature | Description |
|---|---|
| 🤖 **AI Question Generation** | Dynamic questions generated by Google Gemini — never the same game twice |
| 🧠 **Smart Answer Evaluation** | NLP-based semantic matching — accepts partial/paraphrased answers |
| 📈 **Adaptive Difficulty** | Difficulty scales from Beginner to Expert based on your performance |
| 🎯 **Topic Preferences** | Tracks what you're good at and serves you more of what you enjoy |
| 🎮 **3 Game Modes** | Quiz Mode, Learning Mode (detailed explanations), Challenge Mode (timed) |
| 💡 **AI Hints** | 2 progressive hints per question — vague first, stronger second |
| 📚 **Explanation Engine** | Every answer comes with an explanation + fun fact |
| 🏆 **Leaderboard** | Compete for top scores across all modes |
| 📅 **Daily Challenge** | New themed challenge every day with its own leaderboard |
| 📊 **Stats Dashboard** | Track your progress with radar charts and score history |
| 🔌 **Offline Mode** | Works without an API key using 55+ built-in questions |

## 🚀 Setup Instructions

### 1. Prerequisites

- Python 3.10 or newer
- A Google Gemini API key (free at [aistudio.google.com](https://aistudio.google.com))

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the App

```bash
streamlit run main.py
```

### 4. Configure AI (Optional)

Once the app opens, enter your Gemini API key in the sidebar under **AI Settings**. Without a key, the app runs in offline mode with built-in questions.
